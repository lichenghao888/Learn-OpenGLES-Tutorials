Welcome to the Learn OpenGL ES project home page! This project is a repository for the lessons and tutorials over at http://www.learnopengles.com/ This project is intended to help people learn to develop games and applications using OpenGL ES on Android and WebGL, focusing on the OpenGL ES 2 flavor using shaders. Any comments and feedback are highly encouraged. Here is a list of the available lessons and tutorials: Android Lesson One: Getting Started Lesson Two: Ambient and Diffuse Lighting Lesson Three: Moving to Per-Fragment Lighting The compiled app can be downloaded from the Android market at https://market.android.com/details?id=com.learnopengles.android WebGL Lesson One: Getting Started The WebGL examples can be seen embedded in their lesson pages at http://www.learnopengles.com/category/tutorials/webgl/ Comments and feedback are encouraged. Please feel free to open any issues at this project page, or contact me at http://www.learnopengles.com/
